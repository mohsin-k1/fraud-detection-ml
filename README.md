# Fraud Detection in Financial Transactions Using Machine Learning Algorithms

## Overview

This project was developed as my final-year dissertation for my BSc (Hons) Software Engineering degree at Nottingham Trent University.

The aim of the project was to develop and evaluate machine learning models capable of identifying fraudulent financial transactions. The project involved preprocessing transaction data, engineering relevant features and comparing multiple machine learning algorithms to determine the most effective model for fraud detection.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn

---

## Machine Learning Algorithms

The following algorithms were implemented and evaluated:

- Decision Tree
- Random Forest
- Neural Network

---

## Features

- Data preprocessing and cleaning
- Feature engineering
- Machine learning model training
- Model comparison and evaluation
- Fraud prediction
- Performance analysis

---

## Results

The developed models achieved approximately **92% prediction accuracy**, demonstrating the effectiveness of machine learning techniques for identifying fraudulent financial transactions.
